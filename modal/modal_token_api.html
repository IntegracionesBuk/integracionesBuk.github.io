Creación de token y conexión
 

Debes hacer click para acceder a los ajustes de la plataforma.

Debes ingresar la palabra Accesos API para encontrar la configuración correcta.

Accesos API

Haz click aqui para ingresar a la creación del token

Click aqui para que comencemos a crear el token.

En este punto, es momento de revisar cada uno de los permisos que requiera el endpoint que desees utilizar.

Luego de configurar los permisos para los endpoint que utilizaremos, daremos click en el botón guardar.

Damos click en el icono del ojo para que el token sea desbloqueado y poder copiarlo.

Copiamos el texto y lo llevaremos a la pagina de apidocs. Ha esta pagina llegamos ingresando la palabra apidocs en la url del cliente ejemplo: demo.buk.cl/apidocs.

Debemos hacer click en Autorizar para que ingresemos nuestro token copiado anteriormente.

  • Debemos ver que el país este correcto

Pegamos nuestro token creado.

Hacemos click en Authorize

Volvemos hacer click en autorizar. Esto es para poder validar que figuremos logeados en apidocs.

Podemos ver que el bóton ya dice logout (cerrar sesión) y que en el campo value figura nuestro token.
